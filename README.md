# Sonar

## Fast start

```bash
sonar --url https://nashe1.hostingradio.ru/ultra-256
```

## Available options

```
Usage: sonar [OPTIONS] --url <STREAM_URL>

Options:
      --url <STREAM_URL>            Sets the capture stream URL
      --timeout <STREAM_TIMEOUT>    Sets the seconds for the stream connection timeout [default: 5]
      --path <RECORDS_PATH>         Sets the path to the records directory [default: records]
      --port <MONITOR_PORT_NUMBER>  Sets the monitor server port [default: 3000]
      --log-level <LOG_LEVEL>       Sets the log level [default: info]
  -h, --help                        Print help
  -V, --version                     Print version
```
